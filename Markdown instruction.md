## Списки

Чтобы добавить не нумерованные списки, необходимо пункты выделить (*). Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо просто пронумеровать.
Например вот так:
1. Первый пункт
2. Пункт
3. Пункт
